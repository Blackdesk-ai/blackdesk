# Code of Conduct

## Our Standard

This project aims to maintain a respectful, focused, and technically rigorous community.

Examples of behavior that contribute to a positive environment include:

- giving constructive, specific feedback
- disagreeing on technical grounds without personal attacks
- assuming good intent while still being precise about risks and tradeoffs
- keeping discussions relevant to the project and its users
- respecting different experience levels and backgrounds

Examples of unacceptable behavior include:

- harassment, insults, or hostile personal remarks
- discriminatory language or behavior
- repeated bad-faith arguments or deliberate derailment
- publishing private information without consent
- spam, trolling, or abusive conduct

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that do not align with this Code of Conduct.

## Reporting

Until the public repository is created, report serious conduct problems privately to the maintainer.

## Scope

This Code of Conduct applies to repository discussions, issues, pull requests, and other project-managed communication channels.
